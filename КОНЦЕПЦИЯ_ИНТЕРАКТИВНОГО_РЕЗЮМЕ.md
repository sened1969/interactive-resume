# Концепция интерактивного резюме

## Анализ идеи

### Преимущества интерактивного резюме

1. **Демонстрация T-shaped навыков**
   - Показывает глубокую экспертизу в двух смежных областях (Руководитель проекта и Бизнес-аналитик)
   - Позволяет избежать перегруженности информацией
   - Каждая роль получает свой фокус и акцент

2. **Интерактивность как доказательство компетенций**
   - Само резюме является кейсом, подтверждающим технические навыки
   - Демонстрирует умение создавать современные цифровые продукты
   - Показывает понимание UX/UI принципов

3. **Масштабируемость**
   - Легко добавлять новые разделы и проекты
   - Возможность обновления без полной переверстки
   - Гибкая структура для будущего расширения

4. **Визуализация портфолио**
   - Интерактивные примеры работ (BI-дашборды, схемы процессов)
   - Возможность просмотра проектов в действии
   - Галерея с модальными окнами для детального просмотра

### Риски и решения

1. **ATS-системы (роботы рекрутеров)**
   - **Риск:** Роботы плохо читают интерактивные сайты
   - **Решение:** Кнопка "Скачать PDF" для классической версии резюме

2. **Мобильная версия**
   - **Риск:** Сложные элементы могут некорректно отображаться на мобильных
   - **Решение:** Адаптивный дизайн с медиа-запросами для всех устройств

3. **Скорость загрузки**
   - **Риск:** Большое количество изображений может замедлить загрузку
   - **Решение:** Оптимизация изображений, ленивая загрузка

## Структура интерактивного резюме

### 1. Header (Шапка)
- **Фотография:** Круглый аватар (качественное деловое фото)
- **Имя и должность:** Динамически меняется в зависимости от выбранной роли
- **Переключатель ролей:** Главный элемент управления (Toggle Switch)
  - Руководитель проекта
  - Бизнес-аналитик

### 2. Navigation (Навигация)
Липкое меню с якорными ссылками:
- Обо мне
- Компетенции
- Опыт
- Портфолио
- Документы
- СМИ
- Контакты

### 3. Hero Section (Обо мне)
- Динамический текст summary, меняющийся при переключении роли
- Краткое описание экспертизы для каждой роли

### 4. Skills Section (Компетенции)
- **Для РП:** PMBOK, Agile, Waterfall, Управление рисками, Бюджетирование, Confluence/Jira
- **Для БА:** BPMN/IDEF0, SQL, Python (Pandas), UML, Разработка ТЗ, Swagger (API)
- **Общие:** 1С:Предприятие, MS Office, Visio
- Визуализация в виде тегов с цветовой дифференциацией

### 5. Experience Section (Опыт работы)
- Хронологическая лента (Timeline)
- Каждая карточка опыта имеет два слоя описания:
  - В режиме РП: акцент на управление, бюджеты, команды, достижения
  - В режиме БА: акцент на анализ, моделирование, разработку ТЗ, автоматизацию

### 6. Portfolio Section (Портфолио)
Интерактивная галерея проектов:
- **BI-аналитика: Ресторан** - Streamlit дашборд
- **BI-аналитика: Стройка** - Streamlit дашборд
- **AI-ассистент закупок** - Telegram-бот
- **Моделирование бизнес-процессов** - BPMN диаграммы
- **Стратегия развития компании** - Стратегическое планирование

Каждый проект содержит:
- Превью изображение
- Краткое описание
- Ссылку на детальный просмотр

### 7. Documents Section (Документы)
- Диплом (будет добавлен позже)
- Сертификат 1С: Управление проектами (PDF)
- Сертификаты дополнительного образования (будут добавлены позже)

### 8. Media Section (Публикации в СМИ)
Список статей и интервью с группировкой по годам:
- 2014: Статья "Настроились на полимеры" (ВСП)
- 2014: Байкальская венчурная ярмарка
- 2013: Конференция "Механика-2013"

### 9. Contact Section (Контакты)
- Email
- Telegram
- Телефон

### 10. CTA Section (Призыв к действию)
- Кнопка "Написать на email"
- Кнопка "Написать в Telegram"
- Кнопка "Скачать PDF"

## Перечень активных элементов и функций

### Интерактивные элементы:

1. **Global Role Toggle (Переключатель ролей)**
   - Перестраивает контент страницы без перезагрузки
   - Меняет заголовки, описания, навыки и опыт работы
   - Плавная анимация переключения

2. **Navigation Bar (Липкое меню)**
   - Плавная прокрутка к разделам
   - Подсветка активного раздела при скролле
   - Адаптивное меню для мобильных устройств

3. **Portfolio Items (Карточки портфолио)**
   - Hover-эффекты при наведении
   - Модальные окна для просмотра изображений в полном размере
   - Ссылки на детальные страницы проектов

4. **Image Modal (Модальное окно для изображений)**
   - Просмотр изображений в полном размере
   - Закрытие по клику вне изображения или на крестик
   - Поддержка клавиатурной навигации (ESC)

5. **Timeline Items (Элементы временной линии)**
   - Визуальная линия времени
   - Карточки опыта с детальным описанием
   - Разные описания для разных ролей

6. **Skill Tags (Теги навыков)**
   - Цветовая дифференциация (общие/специфичные навыки)
   - Визуальное представление компетенций

7. **Document Cards (Карточки документов)**
   - Hover-эффекты
   - Прямые ссылки на PDF файлы
   - Индикация недоступных документов

8. **Media Items (Публикации)**
   - Карточки с описанием
   - Прямые ссылки на источники
   - Группировка по годам

9. **CTA Buttons (Кнопки призыва к действию)**
   - Выделенные кнопки для связи
   - Функция печати/сохранения в PDF
   - Прямые ссылки на мессенджеры

## Технические особенности

### Технологии:
- **HTML5** - семантическая разметка
- **CSS3** - современные стили с переменными, градиентами, анимациями
- **Vanilla JavaScript** - логика переключения ролей и интерактивности
- **Адаптивный дизайн** - медиа-запросы для всех устройств

### Производительность:
- Минимальное использование внешних библиотек
- Оптимизированные изображения
- Плавные анимации через CSS transitions
- Ленивая загрузка изображений (можно добавить)

### Доступность:
- Семантические HTML-теги
- Alt-тексты для изображений
- Клавиатурная навигация
- Контрастные цвета для читаемости

### Печать:
- Специальные стили для печати
- Отображение обеих ролей при печати
- Скрытие навигации и CTA при печати

## Рекомендации по использованию

### Перед публикацией:

1. **Заполните контактные данные:**
   - Email
   - Telegram username
   - Телефон

2. **Добавьте фотографию:**
   - Убедитесь, что файл `Фото/IMG_7791_resize.JPG` существует
   - Или обновите путь к фотографии в HTML

3. **Проверьте ссылки:**
   - Все ссылки на проекты работают
   - PDF файлы доступны
   - Внешние ссылки на СМИ актуальны

4. **Оптимизируйте изображения:**
   - Сожмите изображения для веба
   - Используйте современные форматы (WebP)
   - Добавьте lazy loading

5. **Добавьте мета-теги:**
   - Open Graph для соцсетей
   - Twitter Cards
   - Favicon

### Дополнительные улучшения (опционально):

1. **Аналитика:**
   - Google Analytics для отслеживания посещений
   - События для переключения ролей

2. **SEO:**
   - Структурированные данные (Schema.org)
   - Sitemap.xml
   - Robots.txt

3. **Интеграции:**
   - Форма обратной связи
   - Календарь для записи на встречу
   - Интеграция с LinkedIn

4. **Многоязычность:**
   - Переключение языка (русский/английский)
   - Локализация контента

## Заключение

Интерактивное резюме представляет собой современный подход к презентации профессионального профиля. Оно не только демонстрирует навыки и опыт, но и само по себе является доказательством технических компетенций. 

Ключевое преимущество - возможность динамического переключения между ролями, что позволяет рекрутеру видеть именно ту информацию, которая ему нужна, не теряя при этом глубины профессионального бэкграунда.

Резюме готово к использованию и может быть легко адаптировано под конкретные потребности и дополнено новыми разделами и проектами.
